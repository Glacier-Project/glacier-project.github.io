Glacier Project



About the Glacier Project

The Glacier Project is an effort to build a usable open-source mobile computing platform. Our goal is to create:
  • An open-hardware mobile phone, that can be assembled with readily available parts and tools
  • A secure (but open) mobile operating system
  • Web services (for GPS, IM, etc...) that respect the user's freedom and privacy
Two of these are already being worked on (see below).

Current Projects

Glacier One: A set of reference designs for creating mobile phones.
Glacier OS: An open-source mobile OS, optimized for "dumbphones" with monochrome screens.
